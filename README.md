Graphics Engine is a graphics rendering engine that supports rendering 3 Dimensional openGL graphics and user input. Made in C# using [OpenTK](https://github.com/opentk/opentk).

### Features
- 2D/3D rendering
- Basic lighting
- User input (Keyboard & Mouse)
